# 🎬 MovieLens Analytics Dashboard

## 📖 Project Overview

The MovieLens Analytics Dashboard is an interactive business intelligence application built with **Python**, **Streamlit**, **Pandas**, and **Plotly**.

The dashboard analyzes the MovieLens Latest Small dataset to provide insights into movie ratings, genre popularity, user activity, and movie release trends.

---

## 🎯 Project Objectives

The goal of this project is to:

- Analyze movie ratings
- Identify highly rated and popular movies
- Explore genre distribution
- Understand user engagement
- Analyze movie release trends
- Provide business recommendations based on data

---

## 🛠 Technologies Used

- Python
- Streamlit
- Pandas
- Plotly Express

---

## 📂 Dataset

MovieLens Latest Small Dataset

Source:
https://grouplens.org/datasets/movielens/

Files used:

- movies.csv
- ratings.csv
- links.csv
- tags.csv

---

## 📈 Business Insights

The analysis revealed that:

- Drama, Comedy, and Action dominate the movie catalog.
- A small number of movies receive a large percentage of user ratings.
- Highly rated movies with many ratings should be promoted.
- Low-visibility movies can benefit from improved recommendation algorithms.
